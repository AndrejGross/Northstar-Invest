# Northstar Invest Web

Next.js frontend for the Northstar Invest monorepo.

## Development

From the repository root:

```powershell
pnpm web:dev
```

The app runs at [http://localhost:3000](http://localhost:3000).

## Scripts

```powershell
pnpm web:dev
pnpm web:build
pnpm web:lint
```

The backend API lives in `apps/api`.
